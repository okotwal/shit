Algebr.ai

AI-Powered Algebra Practice Platform

Project Journey

Algebr.ai was envisioned as a platform to simplify and personalize Algebra practice for students. The project started with the goal of making math accessible and adaptive using AI technologies.

Key Learning Moments

  • Early Development: Learning Django as the backend framework to handle dynamic content and integrate AI models effectively.
  • AI Models: Experimented with various AI models to find out which ones would be the best for our project.
  • User Feedback: Conducted user testing sessions to ensure the platform met students' learning needs and adjusted features accordingly.

Challenges and Solutions

  • Challenge: Designing a user-friendly interface for the generated tests.
    Solution: Created a seamless UI using Django templates and HTML/CSS to enhance user experience.
  • Challenge: Correctly prompting our AI models to give the best personlized questions.
    Solution: Tested out many different prompts by testing with fellow peers to see what worked best.

Research and Resources

  • Documentation: Django's official documentation and tutorials for setting up the backend.
  • Udemy:Udemy had some helpful Django courses to get comfortable with the new backend framework.
